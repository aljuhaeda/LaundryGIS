Kenapa Pakai LaundryUIN?

LaundryUIN memetakan 28 layanan laundry nyata di sekitar UIN Malang — lengkap dengan alamat, harga per kilogram, dan estimasi waktu pengerjaan — sehingga mahasiswa bisa membandingkan pilihan sebelum berangkat, bukan menebak-nebak.

Seluruh peta, tabel data, dan dashboard kategori berjalan langsung di browser tanpa server tambahan, jadi selalu bisa diakses kapan saja.

Lihat Peta
28 Lokasi Terpetakan
Setiap lokasi dicatat dengan koordinat asli hasil survei di sekitar UIN Malang
Harga Transparan
Harga per kilogram ditampilkan apa adanya, tanpa perlu menghubungi tiap laundry satu per satu
Filter Kecepatan Pengerjaan
Bandingkan laundry cepat (≤ 6 jam) dan standar (> 6 jam) langsung dari dashboard kategori
Pencarian Cepat
Cari berdasarkan nama atau alamat langsung dari tabel data laundry
Tanpa Server Tambahan
Peta dan dashboard berjalan langsung di browser, tidak butuh instalasi backend untuk digunakan
Proyek Tugas Akhir GIS
Dibangun dengan QGIS untuk pengolahan data spasial, lalu disederhanakan menjadi aplikasi web mandiri
